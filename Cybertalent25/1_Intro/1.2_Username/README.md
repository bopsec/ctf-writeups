# Username

Gjør deg kjent med username-kommandoen. Denne bruker du for å se eller endre
brukernavnet som vises på scoreboardet på nettsiden.

Nettside: https://ctf.cybertalent.no/

```sh
login@corax ~ $ username --help
# Endre navn som vises på scoreboard
#
# Usage: username [OPTIONS] [INPUT]
#
# Arguments:
#   [INPUT]  Egendefinert brukernavn eller ingenting for å lese ut nåværende brukernavn [default: ]
#
# Options:
#       --random  Sett et tilfeldig generert brukernavn
#   -h, --help    Print help
#
login@corax ~ $ username
# Nåværende brukernavn: JOVIAL PROBESNOKER
#
login@corax ~ $ username cybertalent
# Nytt brukernavn: cybertalent
#
login@corax ~ $ username --random
# Nytt brukernavn: SIGNALSTERK ANALYTIKER
#
```

Når du føler deg fornøyd kan du levere dette flagget
for å fullføre oppgaven: `80d32d56199a49d90fd489d139defd0c`
